# 💰 我的記帳 APP (Money Tracker)

這是一個使用 **React Native + Expo** 開發的記帳應用程式，介面高度還原經典記帳 APP 風格。

## 🚀 如何在您的電腦上啟動

如果您發現網頁沒有自動更新，請嘗試以下步驟手動啟動：

1.  **開啟終端機 (Terminal/PowerShell)**
2.  **進入專案目錄**：
    ```bash
    cd "c:\Users\User\Desktop\新增資料夾"
    ```
3.  **啟動 Expo 伺服器**：
    ```bash
    npx expo start --web
    ```
4.  **開啟網頁**：
    在瀏覽器輸入以下網址：
    👉 [http://localhost:8081](http://localhost:8081)

## 💡 常見問題
- **畫面沒變化？** 請在瀏覽器按下 `Ctrl + F5` (硬體重新整理) 強制讀取最新程式碼。
- **想要在手機看？** 下載手機 App **"Expo Go"**，並掃描終端機出現的 QR Code 即可（需在同一 WiFi 下）。
